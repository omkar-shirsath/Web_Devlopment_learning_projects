# Photography Landing Page with Sidebar Navigation

A responsive photography landing page featuring a CSS-only toggle sidebar.

## Features
- Sidebar navigation using pure CSS (checkbox hack)
- Hero section with background image
- Font Awesome icons
- Smooth transitions and hover effects

## Tech Stack
- HTML5
- CSS3 (Flexbox, Transitions, Pseudo-selectors)

## Key Learnings
- UI state handling without JavaScript
- Sidebar animations and layout control
- CSS selector management

## Future Improvements
- Improve mobile responsiveness
- Add JavaScript-based navigation
- Convert layout to React
