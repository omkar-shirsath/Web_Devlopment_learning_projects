# Spotify Web Player Clone

A responsive Spotify Web Player UI clone built using HTML and CSS.

## Features
- Sidebar navigation
- Sticky top navigation bar
- Card-based music sections
- Responsive layout using Flexbox
- Hover interactions

## Tech Stack
- HTML5
- CSS3 (Flexbox, Media Queries)

## Purpose
Built to practice UI layout, responsiveness, and component structuring.

## Future Improvements
- Add JavaScript for dynamic data rendering
- Convert to React
- Add audio playback functionality
